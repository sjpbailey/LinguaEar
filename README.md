# App Information only for public information

Standard layout:

- **Swift/** — Xcode projects
- **Python/** — Python projects
  - **envs/** — one virtual environment per project
  - **Projects/** — per-project source folders
  - **Scripts/** — utility scripts
- **VSCode/** — editor settings (optional)
- **Docs/** — tutorials, notes, PDFs
- **GitHub/** — working clones / mirrors
- **XcodeDerivedData/** — optional custom DerivedData path
