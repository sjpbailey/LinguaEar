## ✅ Newly Added Features (Version 2.0)

### 1. Listen & Repeat Practice — Segmented Playback (NEW)
- Practice mode now supports **sentence-level playback** for long phrases or paragraphs
- Text is automatically split into sentence-like segments
- Users control playback with transport buttons:
  - ⏮ Back — previous sentence
  - ▶️ Play — play current sentence
  - ⏸ Pause — pause speech
  - ⏹ Stop — stop playback
  - ⏭ Forward — next sentence
- Enables controlled listening for languages without clear word boundaries (e.g. Japanese, Chinese)
- Works with both preset phrases and custom typed phrases

### 2. Compact Audio Transport UI (NEW)
- Replaced single “Listen” button with compact transport bar
- Transport bar appears below speed controls
- “Repeat” (microphone) remains separate for speech scoring
- Improves usability for long listening sessions and learning workflows
